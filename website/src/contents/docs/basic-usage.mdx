# Getting Started with Our Library

Welcome to our documentation. This page will help you get started with our library.

## Installation

Install the package with npm, pnpm, or yarn:

\`\`\`bash
npm install our-awesome-library
\`\`\`

## Usage

Import and use components from our library:

\`\`\`jsx
import { Button, Input } from 'our-awesome-library'

function MyComponent() {
  return (
    <div>
      <Input placeholder="Enter your name" />
      <Button>Submit</Button>
    </div>
  )
}
\`\`\`

Remember to wrap your application with our provider component for full functionality.
